## Overview

Novagent is a fully autonomous AI agency that delivers professional services across video, design, audio, software, research, data, content, business operations, translation, education, tech support, and more. It operates 24/7, finds its own clients, generates business documents for every job, and maintains a three-layer security stack.

- **48 specialized skills** working in orchestration
- **~116 services** across **12 categories**
- **Autonomous lead prospecting** and outreach
- **Full business documentation** (proposals, invoices, receipts, reports, manuals)
- **Real-time dashboard** and financial tracking
- **Enterprise-grade security** with kill switch

## Security Architecture

Novagent has a **three-layer security stack** — no other AI agent in production has this depth.

### Layer 1 — NemoClaw OpenShell (Kernel)
- Kernel-level sandbox isolation
- Credential proxy injection
- Filesystem and network policy enforcement
- Prevents agent escape

### Layer 2 — Tool Trust (Application)
- Every tool call verified against a signed **trust passport** before execution
- **4 trust tiers:** Fully Trusted → Verified External → Community Tools → Unknown (blocked)
- Pre-call verification checklist (6 checks)
- Tool call logging for every Tier 2+ action
- Trust revocation when tools fail or exceed cost estimates

### Layer 3 — Spend Gates (Financial)
- **4 gate levels:** Micro ($0-1) → Standard ($1-10) → Significant ($10-50) → Major ($50+)
- Daily spend cap: **$200**
- Per-job budget tracking with margin alerts
- Hard stop at 80% margin erosion
- Kill switch integration for instant spending freeze

### Kill Switch
Trigger words: `KILL`, `STOP`, `HALT`, `SHUTDOWN`, `FREEZE`, `ABORT`
- Immediate freeze of all spending and API calls
- Stop all sub-agents
- Save all state
- Report full status within 60 seconds
- Resume with `RESUME` — everything recovers
- Partial kills available: `STOP SPENDING`, `STOP OUTREACH`, `STOP CRONS`, `STOP CLIENTS`

## Business Documentation System

Every job generates a complete paper trail. Documents are generated as formatted messages or downloadable HTML files.

### Sent to Client:
| Document | When | Purpose |
|----------|------|---------|
| **PROPOSAL** | After scoping | Scope, pricing, timeline, data handling commitment |
| **INVOICE** | After approval | Itemized bill with Stripe payment link |
| **RECEIPT** | After payment | Proof of payment with transaction ID |
| **USER MANUAL** | At delivery (technical jobs) | Instructions for using what was built |
| **THANK YOU** | After delivery confirmed | Warm closing with referral offer |

### Sent to Operator (internal):
| Document | When | Purpose |
|----------|------|---------|
| **JOB REPORT** | After delivery | Internal metrics, costs, margin, methodology |
| **AUDIT CERTIFICATE** | After cleanup | Security log, tools used, data handling proof |

### Document ID Format
`[TYPE]-[YYYYMMDD]-[SEQ]` — e.g., `PROP-20260627-001`, `INV-20260627-001`

## Core Workflow

### 17-Step Job Lifecycle
1. Receive client request via Telegram/Discord/WhatsApp/SMS/Website/Phone
2. If new client → run **smart-onboarding** (free sample + preview process)
3. Present NDA/data handling commitment (**nda-compliance**)
4. Scope work using **reasoning-engine** (complexity, tools, time, cost)
5. Generate **PROPOSAL** using **business-docs** and send to client
6. Quote price using **dynamic-pricing** (cost × 3.5 + complexity premium)
7. After approval → generate **INVOICE** with Stripe payment link
8. After payment confirms → generate **RECEIPT**
9. Log decision in **decision-audit** with full reasoning
10. Show client previews at each stage (**client-preview**)
11. For complex jobs, delegate to specialist sub-agents (**job-dispatcher**)
12. Deliver work → generate **JOB REPORT** + **USER MANUAL** (if technical)
13. Send **THANK YOU** message with referral offer
14. Run **vault-cleanup** (audit certificate, sanitize memory, purge data)
15. Log job in P&L (**pnl-tracker**)
16. Update **dashboard-state.json** via **dashboard-updater**
17. 24h later → **client-feedback** → if 4-5 stars → **referral-system**

### Parallel Systems (always running)
- **lead-prospector:** Daily search for new business opportunities (8 AM)
- **outreach-engine:** Personalized outreach to qualified leads
- **resilience:** Health check every 6 hours
- **mission-control:** Dashboard auto-update every 30 minutes
- **tool-trust:** Verification before every external tool call
- **spend-gates:** Cost check before every financial action
