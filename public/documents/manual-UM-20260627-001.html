1. QUICK START

  1. Open your terminal and navigate to the project: cd ~/acme-dashboard
  2. Run the integration script: python3 analytics_widget.py --config config.yaml
  3. Open your browser to http://localhost:8080
  4. The analytics widget will appear on your dashboard

2. WHAT WAS BUILT

  • analytics_widget.py: Python script - Pulls data from your API and renders as interactive chart
  • config.yaml: Configuration file at ~/acme-dashboard/config.yaml
  • requirements.txt: Python dependencies (pandas, plotly, flask)
  • Dockerfile: Container setup for production deployment

3. HOW TO USE IT

Step 1: Configure your API key in config.yaml

Step 2: Run pip install -r requirements.txt

Step 3: Run python3 analytics_widget.py

Step 4: Access the dashboard at http://localhost:8080

4. CONFIGURATION OPTIONS

  • api_key: Your Acme API key (required)
  • refresh_interval: Data refresh in seconds (default: 300)
  • theme: 'light' or 'dark' (default: dark)

6. TROUBLESHOOTING

Problem: Widget shows no data
Solution: Check your API key in config.yaml and verify network connectivity

Problem: Port 8080 is in use
Solution: Change port in config.yaml: port: 8081

7. FAQ

Q: Can I deploy this to production?
A: Yes - use the included Dockerfile: docker build -t acme-widget . && docker run -p 8080:8080 acme-widget

Q: Does it support multiple users?
A: Yes - set multi_user: true in config.yaml
